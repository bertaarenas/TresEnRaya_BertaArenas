# TresEnRaya_BertaArenas

Este programa es un juego de Tres en Raya para dos jugadores.
El juego tiene un menú donde se puede jugar, cambiar la configuración o salir. 
Primero se introducen los nombres de los jugadores y el programa elige al azar quién empieza.
En cada turno el jugador indica la fila y la columna donde quiere poner su ficha y gana quien complete una fila, columna o diagonal.
El tamaño del tablero y el número de partidas para ganar se pueden cambiar yendo al segundo punto del menú.
Por último, dandole al 3 en el menú podras salir de la partida y el juego mostrara el total de partidas ganadas de cada jugador.
